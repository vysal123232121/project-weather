🌦️ Project Weather
A sleek, responsive application that provides real-time weather updates, forecasts, and environmental data. 
Whether you're planning a hike or just wondering if you need an umbrella, Project Weather has you covered.

🚀 Features
# Real-time Data: Fetch current temperature, humidity, and wind speed for any city.
# 5-Day Forecast: Plan ahead with daily high/low predictions.
# Dynamic Backgrounds: The UI changes based on the weather (sunny, rainy, snowy).
# Geolocation: Automatically detects your local weather upon permission.
# Unit Conversion: Toggle between Metric (°C) and Imperial (°F).
